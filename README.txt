
How to publish (Netlify Drag & Drop)
------------------------------------
1) Open this 'site_upload_ready' folder.
2) You should see index.html and other pages inside.
3) Go to netlify.com → Add new site → Deploy manually.
4) Drag this entire folder into the drop zone.
5) You’ll get a live link instantly.

Other hosts:
- Vercel: import as 'static' project (no build step).
- GitHub Pages: push this folder to a repo and enable Pages.
- cPanel: upload this folder to public_html and Extract.

Wix note: Wix does not accept full HTML sites. Rebuild in Wix editor or embed pages.
